Chat with any PDF, fully in your browser

Nothing leaves your machine — the language model and the document index both run locally via WebAssembly/WebGPU. Load a PDF to begin.

First load downloads the model (~2 GB, cached afterwards). A WebGPU-capable browser (Chrome/Edge) is recommended. Documents are indexed once and cached locally, so re-opening the same PDF is instant.
